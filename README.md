# Abby
A lightweight HTML/CSS boilerplate, built with SCSS.

### NPM
```
npm install abby --save-dev
```

### Use CSS
```html
<link rel="stylesheet" href="abby.css">
```

### Use SCSS
```css
@import "abby";
```
