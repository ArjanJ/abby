Typography

Abby's typography follows a simple set of rules, all in rem units. To create a consistent vertical rhythm, all headings and paragraphs have the same margin-bottom value, which is equal to the body font size multiplied by the base line-height. The font sizes throughout Abby are calculated by following the perfect fith ratio to create a consistent and pleasing look.

Grids

Abby's grid system is created using flexbox. The markup and flexibility is dead simple. Simply add an element with the class grid__col inside of an element with the class grid and it will be treated as a column. There are no container or row classes, simply a grid class which is a container for the grid columns. To create a new row you must create a new element with the grid class.

.grid__col .grid__col--3
.grid__col
.grid__col .grid__col--2
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col
.grid__col

Example Markup:

<div class="grid">
  <div class="grid__col">
  <div class="grid__col">
</div>

Modifier Classes

In order to have columns that are larger than others in the same "row" we need some modifier classes to change the flex-grow property. The class grid__col--3 means make this element 3x as large as the other elements in the same "flexed" box. Within Abby there are six modifier classes available for the grid__col class.

Modifier Class Name Flex-grow Value
grid__col--6 6
grid__col--5 5
grid__col--4 4
grid__col--3 3
grid__col--2 2
grid__col--1 1

Gutters

Abby has customizable gutters. By default the grid gutters are set to 0.8rem. This can be changed in /src/_variables.scss.

Forms

Utilities

Abby comes with a few utilies that can help speed things up when starting a project. Some features are only available to use with SCSS, which is recommended.

Mixins

Breakpoints

Abby provides some media query mixins to helpout with responsive design. These are the default breakpoints, which are configurable in src/_variables.scss.

Variable Name Value (px)
$bp-xsmall 480
$bp-small 640
$bp-medium 768
$bp-large 960
$bp-xlarge 1100

The mixins for each breakpoint are as follows.

@mixin bp-custom($width) {
  @media screen and (min-width: $width) {
    @content;
  }
}

@mixin bp-xsmall {
  @media screen and (min-width: $bp-xsmall) {
    @content;
  }
}

@mixin bp-small {
  @media screen and (min-width: $bp-small) {
    @content;
  }
}

@mixin bp-medium {
  @media screen and (min-width: $bp-medium) {
    @content;
  }
}

@mixin bp-large {
  @media screen and (min-width: $bp-large) {
    @content;
  }
}

@mixin bp-xlarge {
  @media screen and (min-width: $bp-xlarge) {
    @content;
  }
}

Clearfix

To clearfix an element you can use @include clearfix in your stylesheet, or you can apply the clearfix class in your HTML.

Transitions

Abby includes a mixin to help with the transition property. Simply specify the easing type ('in', 'out', 'in-out') and the speed. The defaults can be changed in src/_variables.scss.

Example:

.element {
  @include ease('in', 0.5);
  color: red;
}
